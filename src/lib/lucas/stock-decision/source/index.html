新旧凯利公式推导

旧公式：赔率版凯利

适用场景：每次投入总资产比例 \(f\)，成功时每 1 元投注净赚 \(b\) 元，失败时这 1 元投注全部亏掉。

\[ \begin{aligned} p &= \text{胜率} \\[0.45em] q &= 1 - p \\[0.45em] b &= \text{净赔率} \\[0.45em] f &= \text{投入总资产的比例} \end{aligned} \]

胜利后的资产倍数为：

\[ 1 + f b \]

失败后的资产倍数为：

\[ 1 - f \]

长期增长率为：

\[ g(f) = p \ln\left(1 + f b\right) + q \ln\left(1 - f\right) \]

对 \(f\) 求导：

\[ g'(f) = \frac{p b}{1 + f b} - \frac{q}{1 - f} \]

令 \(g'(f) = 0\)：

\[ \begin{aligned} \frac{p b}{1 + f b} &= \frac{q}{1 - f} \\[0.45em] p b(1 - f) &= q(1 + f b) \\[0.45em] p b - p b f &= q + q f b \\[0.45em] p b - q &= f b(p + q) \end{aligned} \]

因为 \(p + q = 1\)，所以：

\[ \begin{aligned} f^* &= \frac{b p - q}{b} \\[0.45em] &= p - \frac{q}{b} \end{aligned} \]

因此，旧赔率版凯利公式为：

\[ f^* = p - \frac{q}{b} \]

新公式：收益幅度版凯利

适用场景：股票交易中，失败通常不是归零，而是下跌到止损线。设成功收益幅度为 \(r_{\text{win}}\)，失败亏损幅度为 \(r_{\text{loss}}\)。

\[ \begin{aligned} p &= \text{胜率} \\[0.45em] q &= 1 - p \\[0.45em] r_{\text{win}} &= \text{成功时收益幅度} \\[0.45em] r_{\text{loss}} &= \text{失败时亏损幅度} \\[0.45em] f &= \text{投入总资产的比例} \end{aligned} \]

胜利后的资产倍数为：

\[ 1 + f \cdot r_{\text{win}} \]

失败后的资产倍数为：

\[ 1 - f \cdot r_{\text{loss}} \]

长期增长率为：

\[ g(f) = p \ln\left(1 + f r_{\text{win}}\right) + q \ln\left(1 - f r_{\text{loss}}\right) \]

对 \(f\) 求导：

\[ g'(f) = \frac{p r_{\text{win}}}{1 + f r_{\text{win}}} - \frac{q r_{\text{loss}}}{1 - f r_{\text{loss}}} \]

令 \(g'(f) = 0\)：

\[ \begin{aligned} \frac{p r_{\text{win}}}{1 + f r_{\text{win}}} &= \frac{q r_{\text{loss}}}{1 - f r_{\text{loss}}} \\[0.45em] p r_{\text{win}}(1 - f r_{\text{loss}}) &= q r_{\text{loss}}(1 + f r_{\text{win}}) \\[0.45em] p r_{\text{win}} - p f r_{\text{win}} r_{\text{loss}} &= q r_{\text{loss}} + q f r_{\text{win}} r_{\text{loss}} \\[0.45em] p r_{\text{win}} - q r_{\text{loss}} &= f r_{\text{win}} r_{\text{loss}}(p + q) \end{aligned} \]

因为 \(p + q = 1\)，所以：

\[ \begin{aligned} f^* &= \frac{p r_{\text{win}} - q r_{\text{loss}}}{r_{\text{win}} r_{\text{loss}}} \\[0.45em] &= \frac{p}{r_{\text{loss}}} - \frac{q}{r_{\text{win}}} \end{aligned} \]

因此，收益幅度版凯利公式为：

\[ f^* = \frac{p}{r_{\text{loss}}} - \frac{q}{r_{\text{win}}} \]

两个公式的关系

旧公式是新公式在 \(r_{\text{loss}} = 1\)、\(r_{\text{win}} = b\) 时的特殊情况：

\[ \begin{aligned} f^* &= \frac{p}{r_{\text{loss}}} - \frac{q}{r_{\text{win}}} \\[0.45em] &= \frac{p}{1} - \frac{q}{b} \\[0.45em] &= p - \frac{q}{b} \end{aligned} \]
\[ \boxed{ \text{赔率下注使用 } f^* = p - \frac{q}{b} } \] \[ \boxed{ \text{股票涨跌幅使用 } f^* = \frac{p}{r_{\text{loss}}} - \frac{q}{r_{\text{win}}} } \]
