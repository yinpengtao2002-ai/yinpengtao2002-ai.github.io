计算逻辑 (当前维度口径)

  1. 当前维度聚合: 每张图直接按正在查看的维度汇总销量和边际，例如大区、国家或车型。
  2. 结构效应 (Mix): 当前维度销量占比变化带来的单车边际影响。
  3. 费率效应 (Rate): 当前维度自身单车边际变化带来的影响。
  4. 口径差异: 不同维度下 Mix 和 Rate 的拆分会不同，这是聚合口径不同造成的正常结果。
💡 当前口径更适合国家、车型、版型频繁变化的数据。新增或退出的维度项缺少可比单车边际，因此归入结构效应；总贡献仍回到该视图的单车边际变化。

📐 当前维度口径说明

本工具按当前展示维度直接计算归因，不再强制从最底层组合向上汇总。这样能减少国际业务中版型、国家、配置频繁变化带来的不可比问题；如果需要解释更底层的根因，请继续下钻到对应维度观察。

维度项类型 结构效应 (Mix Effect) 费率效应 (Rate Effect)
可比维度项 (当期权重 - 基期权重) × (基期单车边际 - 基期平均单车边际) 当期权重 × (当期单车边际 - 基期单车边际)
新增维度项 当期权重 × (当期单车边际 - 基期平均单车边际) 0
退出维度项 - 基期权重 × (基期单车边际 - 基期平均单车边际) 0

💡 阅读提示：大区、国家、车型等不同维度的结构效应和费率效应总和可能不同，这不是计算错误，而是当前维度口径下的业务解释不同。判断一个异常时，先看当前维度是否能回答问题，再决定是否继续下钻。
